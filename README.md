# CareFlow - Sistema de Cadastro e Prontuários

Projeto de conclusão desenvolvido para o Programa de Desenvolvedor Trainee 2026 da BGMax Tecnologia. O sistema gerencia o cadastro de pacientes e o histórico de seus respectivos atendimentos.

##  Tecnologias Utilizadas
- Python 3.14.2
- Django Framework
- SQLite (Banco de dados integrado)
- Bootstrap 5 (Interface responsiva)
